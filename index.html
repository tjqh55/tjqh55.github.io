MarkDown GitHub사용해서 블로그 올리기

Mac에서는

GitHub 로그인 후
create new repository

Repository name 부분에 자신의id.github.io.git으로 하고 Create repository를한다.

노드가 없다면 설치

노드 설치후 hexo-cli를 ( -g 옵션: 글로벌하게 사용) 사용하기 위해 설치

git clone https://github.com/tjqh55/tjqh55.github.io.git 해당디렉토리

echo “Hello World” >> index.html

git add .

git commit -m ‘firstpush’

git push origin master

hexo init 해당디렉토리

language : ko-KR
timezone : Asia/Seoul
url: http://자신의ID.github.io
deploy:
type : git
repo : git Address

hexo server

npm install –save hexo-deployer-git

hexo deploy –generate 명령어를 사용해 git에 올릴 파일을 만든다.

theme : 자신의 폰트 이름

hexo deploy –generate

hexo new draft first -> .md파일을 생성(source디렉토리 밑)

hexo publish md파일 이름
hexo deploy –generate
