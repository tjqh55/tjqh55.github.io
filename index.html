hexo사용법

MarkDown GitHub사용해서 블로그 올리기

Mac에서는

GitHub 로그인 후
create new repository

Repository name 부분에 자신의id.github.io.git으로 하고 Create repository를한다.

노드가 없다면 설치

노드 설치후 hexo-cli를 ( -g 옵션: 글로벌하게 사용) 사용하기 위해 설치

해당 디렉토리로 이동을 한 후 test를 하기위해 index.html파일에 Hello World를 저장

이제 index.html를 git에 올릴 차례

git 주소(자신의ID.github.io)로 접속하면 Hello World가 뜬다.

express로 해당디렉토리를 설정한다.

해당 디렉토리 이동을 하게되면 여러 파일이 생긴걸 볼수 있다.
여러개의 파일 중 _config.yml에 들어가서 이 부분을 고쳐준다.

고쳐준 뒤에

명령어 친후 chrome에서 localhost:4000접속

설치한다.

예전 기록이 뜨면 방문기록 삭제 후 블로그 주소 다시 접속
