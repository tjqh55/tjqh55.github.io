MarkDown GitHub사용해서 블로그 올리기

GitHub 로그인 후
create new repository

id.github.io.git

노드가 없다면 설치

npm install hexo-cli -g

ssh
git clone https://github.com/tjqh55/tjqh55.github.io.git 해당디렉토리

echo “Hello World” >> index.html

git add .
git commit -m ‘firstpush’
git push origin master

깃 주소(tjqh55.github.io)로 접속하면 Hello World가 떠짐

hexo init 해당디렉토리

해당 디렉토리 이동후 _config.yml

language : ko-KR
timezone : Asia/Seoul
url: http://tjqh55.github.io
deploy:
type : git
repo : git Address
:wq

hexo server 명령어 친후 chrome에 localhost:4000접속

npm install –save hexo-deployer-git 설치후

hexo deploy –generate 올릴 걸 만듬

예전 기록이 뜨면 방문기록 삭제 후 블로그 주소 다시 접속

테마적용
hexo.io aloha 폰트 변경
theme 접속후 자신의 마음에드는 테마를 선택후 해당디렉토리에서 git clone
_config.yml 파일에 들어가서
theme : 자신의 폰트 이름

hexo deploy –generate

hexo new draft first -> .md파일을 생성(source디렉토리 밑)

포스팅
hexo publish md파일 이름
hexo deploy
hexo generate
